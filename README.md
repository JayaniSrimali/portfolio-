# Portfolio - Full Stack Developer

A modern, responsive portfolio website built with:
- **Next.js** (App Router, Server Components)
- **Tailwind CSS v4** (Utility-first CSS framework with JIT)
- **Framer Motion** (Professional animations)
- **Lucide Icons** (Beautiful icons)
- **TypeScript** (Strict type safety)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
   *(Already installed)*

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure

- `src/app`: Page routes and layouts.
- `src/components`: UI components, layout (Navbar/Footer), sections (Hero, Projects).
- `src/lib`: Utility functions.
- `src/app/globals.css`: Tailwind theme configuration.

## Features

- **Dark Mode**: By default with vibrant neon accents.
- **Glassmorphism**: Using backdrop filters on navigation.
- **Animations**: Entrance and scroll animations using Framer Motion.
- **Responsive**: Mobile-first design using Tailwind CSS.
